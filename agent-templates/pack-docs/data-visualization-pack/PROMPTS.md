# Recommended Prompts

Use these prompts with Tandem to get the most out of this pack:

- "Load the matplotlib template and modify it to show a line chart of my monthly sales data."
- "Using the seaborn template, create a correlation heatmap for the variables in `data.csv`."
- "I need an interactive scatter plot. Adapt the plotly template to visualize `x` vs `y` with `category` as color."
- "How do I change the color palette in the matplotlib template to use my company's brand colors?"
- "Help me debug this error I'm getting when trying to run the seaborn script."
