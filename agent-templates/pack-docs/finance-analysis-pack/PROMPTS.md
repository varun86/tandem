# Recommended Prompts

Use these prompts with Tandem to customize your analysis:

- "Modify the income statement script to include a column for 'Year over Year' growth."
- "I have a different chart of accounts. Help me update the 'structure' dictionary in the P&L generator."
- "Update the variance analysis script to flag variances greater than $5,000 regardless of the percentage."
- "How can I read data from an SQL database instead of a CSV file for these reports?"
- "Create a new script that calculates key financial ratios (Gross Margin, Operating Margin) from the income statement data."
